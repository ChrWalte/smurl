smurl.xyz

smurl is a simple URL shortener that allows users to shorten or encapsulate URLs. Currently, the system is simple and straight to the point but with time the system will become more advanced and provide more features. smurl uses a first come first served slug system where once a slug is obtained, it is registered forever. However, a pruning system is in the works that will delete unused smurls after a set amount of days.
